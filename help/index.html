Slither Help

Slither is a Python module designed for bridging the gap between Scratch and Python. It adds Scratch-like features to Python using Pygame.

If you are new to Slither, make a simple Slither program with this:

                    import slither
                    slither.slitherStage.setColor(40, 222, 40)
                    
                    slither.setup() # Begin slither

                    def loop():
                        pass

                    slither.runMainLoop(loop)

After you run this you'll get a green window.So how does this work? It uses a number of things that I'll explain step by step.

import slither Allows Slither to be used in your code.

slither.slitherStage.setColor() Changes the background color, you enter values in the form of "number, number, number", where number represents a RGB value.

slither.setup() Sets up Slither, similar to "pygame.init()".

def loop(): This function is usually called 60 times a second (depending on your FPS). Your program's main loop should be in here.

pass For this example there is no main loop, so pass skips the main loop.

slither.runMainLoop(loop) This important fuction runs your main loop, you need this for your code to run.
Stage FunctionsBrief description
__init__()Initilizes the stage and sprite costumes. You shouldn't call this manually.
setColor()Sets the stage's background color (Stage Only).
addCostume()Add a costume based on a given path and name.
deleteCostumeByName()Delete a costume by name.
deleteCostumeByNumber()Delete a costume by number.
setCostumeByName()Set a costume by it's name.
setCostumeByNumber()Set a costume by it's number.
getCostumeNumber()Get the costume number of a sprite.
getCostumeName()Get the costume name of a sprite.
Sprite FunctionsBrief description
__init__()Initilizes a sprite.
goto(x, y)Tells the sprite to go to (x, y).
moveSteps(setps)Move (steps) steps forward, based on direction.
isVisible()Check if the object is visible, not just showing.
delete()Remove the sprite from the global sprites list, causing it not to be drawn.
Sprite VariablesBrief description
Sprite.currentCostumeThe costume the sprite is set to (in the form of a Pygame rect).
Sprite.xposThe X position of the sprite.
Sprite.yposThe Y position of the sprite.
Sprite.directionWhich direction the sprite is pointing in. Sprites point to 0 by default.
Sprite.showSet to "True" to show the sprite, set to "False" to hide it. Replaces "show()" and "hide()" from v0.2.2 on.
Sprite.scaleHow big the sprite is. Set to 1 by default.
Sprite.zindexDefault is 0. Determines how high up the sprite is. For example, a sprite with a z-index of 1 will be drawn above a sprite with a z-index of 0.
Other FunctionsBrief description
setup()Sets up PyGame and returns a screen object that can be used with blit().
blit()Draw all graphic changes on the screen. This must be called to display graphics.
setFPS()Set the FPS of the project. Default is 60, and Scratch runs at 30.
registerCallback()Register the function func to handle the event eventname
runQuitCallback()Quit eventname handling.
runMainLoop()Run the main game loop.
